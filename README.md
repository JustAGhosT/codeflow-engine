# CodeFlow Monorepo README

Welcome to the CodeFlow monorepo! This repository contains all core components, extensions, infrastructure, and documentation for the CodeFlow ecosystem.

## Structure

- `engine/`           - Python core engine
- `desktop/`          - Desktop app (Electron/Tauri)
- `vscode-extension/` - VS Code extension
- `website/`          - Documentation and marketing site
- `infrastructure/`   - Kubernetes, Terraform, Docker, etc.
- `templates/`        - Workflow templates
- `tools/`            - Developer tools and scripts
- `tests/`            - Unified test suite
- `docs/`             - Documentation
