# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2025-11-17

### Added

- Created a `CHANGELOG.md` file to track changes between releases.
- Added progress indicators to the `check` and `split` CLI commands.

### Changed

- Updated the project version to `1.0.1` and the development status to "Production/Stable".
- Improved the help messages for the CLI commands.
- Updated the CLI's version to `1.0.1`.

### Fixed

- Resolved Python 3.13 dependency compatibility issues.
- Fixed a bug that was causing the test suite to fail.

### Removed

- Removed the `langchain` integration due to unresolved dependency conflicts.
